# Инструкция: Запуск проекта в Claude Code

## Подготовка

### 1. Убедись, что установлены:
- Node.js (v18+)
- Git
- GitHub CLI (`gh`) — для создания репозитория
- Vercel CLI (`npm i -g vercel`) — для деплоя

### 2. Авторизация (если ещё нет):
```bash
gh auth login        # GitHub
vercel login         # Vercel
```

---

## Запуск в Claude Code

### Шаг 1: Создай папку и скопируй CLAUDE.md

```bash
mkdir erlebnisland-kegelsberg
cd erlebnisland-kegelsberg
```

Скопируй файл `CLAUDE.md` в эту папку (из пакета, который я подготовил).

### Шаг 2: Запусти Claude Code и дай команду

Открой Claude Code в папке проекта и вставь этот промпт:

---

**Промпт для Claude Code:**

```
Прочитай CLAUDE.md в этой директории. Это полная документация проекта.

Твоя задача:
1. Инициализируй Vue 3 + Vite проект в текущей директории
2. Установи все зависимости (vue-router, tailwindcss, @tailwindcss/vite, gsap, @vueuse/core)
3. Создай все 4 страницы и все компоненты по структуре из CLAUDE.md
4. Используй весь контент (тексты, цены, контакты) из CLAUDE.md — всё на немецком
5. Используй Unsplash-ссылки для изображений как указано в CLAUDE.md
6. Реализуй все премиальные дизайн-элементы: GSAP-анимации, параллакс, glass-morphism навбар, scroll reveal, page transitions
7. Убедись что сайт выглядит как сайт за €15,000+ — это должен быть WOW-эффект
8. Добавь vercel.json для SPA routing
9. Проверь что `npm run build` проходит без ошибок
10. Создай GitHub репозиторий: `gh repo create erlebnisland-kegelsberg --public --source=. --remote=origin`
11. Закоммить и запуш всё в GitHub
12. Задеплой на Vercel: `npx vercel --prod`

Делай всё автономно, не останавливайся и не спрашивай. Весь контент и инструкции в CLAUDE.md.
```

---

### Шаг 3: Дождись завершения

Claude Code создаст проект, задеплоит на Vercel и выдаст URL.

### Шаг 4: Проверь результат

Открой выданный Vercel URL и проверь:
- Все 4 страницы работают
- Анимации плавные
- Мобильная версия выглядит хорошо
- Навигация работает
- Изображения загружаются

---

## Если что-то нужно поправить
